# SafeLock

SafeLock is a secure digital vault application developed using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB.

## Features
- User Registration & Login
- OTP Verification
- Secure File Upload
- Photo, Video & Document Vault
- Encryption
- Cloudinary Integration

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB
- Cloudinary
